🔥 Configuração Firebase — Sincronização em Tempo Real

Para que todos vejam os dados em tempo real, você precisa criar um banco de dados gratuito no Firebase. Siga os passos abaixo uma única vez. Depois, qualquer pessoa com o arquivo verá as atualizações instantaneamente.

📋 Passo a passo para criar o Firebase (5 minutos, grátis)

1
Acesse console.firebase.google.com e faça login com sua conta Google
2
Clique em "Adicionar projeto" → dê o nome ssp36-dashboard → desative o Google Analytics → clique em "Criar projeto"
3
No menu lateral clique em "Realtime Database" → "Criar banco de dados" → escolha a região us-central1 → selecione "Iniciar no modo de teste" → Ativar
4
Copie a URL do banco. Exemplo: https://ssp36-dashboard-default-rtdb.firebaseio.com — cole no campo abaixo
5
No menu lateral clique em ⚙ Configurações do projeto → role até "Seus apps" → clique em "</> Web" → dê um nome → copie os valores de apiKey e projectId e cole abaixo
📌 Solução rápida — 2 minutos:

O Firebase exige que o arquivo seja servido por um servidor web (não pode abrir direto do computador).

Opção 1 — Netlify (recomendado):
1. Acesse app.netlify.com/drop
2. Arraste o arquivo dashboard_ssp36_firebase.html para a área indicada
3. Abra o link gerado no navegador — o Firebase vai funcionar normalmente

Opção 2 — VS Code Live Server:
Se tiver o VS Code, instale a extensão Live Server e clique em "Open with Live Server"

Opção 3 — Chrome com flag:
Feche o Chrome → abra o terminal e execute:
chrome --allow-file-access-from-files
Depois abra o arquivo normalmente.

📋 Como usar o painel em tempo real

1

Firebase conectado ✅ — todos que abrirem o arquivo verão os mesmos dados

Cole o texto do Meli e clique "Importar" — aparece para todos em segundos

5
